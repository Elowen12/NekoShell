interactuar con archivos o comandos python

crear una clase
npy -c NombreClase

Ejecutar un script
npy -e NombreScript

crear un entorno virtual
npy -v NombreEntorno

ingresar al entorno virtual
npy -ev NombreEntorno

instalar librerias python
npy -i NombreLibreria

interactuar con Git
Debes de tener git instalado en tu computadora

clonar repositorio
ngit -c linkRepositorio

ver status del repositorio
ngit -s

ver registros del repositorio
ngit -l

agregar contenido al repositorio para pasar luego al commit
ngit -a

realizar un commit al repositorio
ngit -m comentario ejemplo: ngit -m Este es un comentario

subir el repositorio al repositorio remoto
ngit -o NombreRutaRamaMaster

configurar un usuario en git
ngit -u NombreUsuario

configurar un correo en git
ngit -e NombreCorreo
